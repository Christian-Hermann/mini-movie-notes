# Mini Movie Notes

Mini Movie Notes is a small React and Express application I built to refresh and reinforce core frontend and backend concepts.

Live Site:
https://your-netlify-url.netlify.app

API:
https://mini-movie-notes-api.onrender.com

After spending time working on larger projects, certifications, and my job search, I wanted to build a simple application from scratch to review React state, components, props, CRUD operations, and frontend/backend communication before moving on to my next major project, Sip n' Shred.

The app allows users to:

- Add movie notes
- Edit movie notes
- Delete movie notes
- View all saved movie notes

The frontend was built with React and the backend was built with Express. Movie data is currently stored in memory, so it resets whenever the server restarts.

This project gave me additional practice with:

- useState
- useEffect
- Controlled forms
- Components and props
- Passing functions as props
- Express routes
- GET, POST, PUT, and DELETE requests
- Using fetch to communicate between the frontend and backend

The goal of this project was not to build a large application, but to strengthen the fundamentals and continue building after bootcamp while preparing for future projects.
